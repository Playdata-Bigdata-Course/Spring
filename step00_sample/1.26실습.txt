
1. 자바소스
	1. Person.java
		- 순수 데이터 표현용 DTO
	2. RunTest.java
		- main() 보유
		- 설정 파일을 인식해서 Person 객체 생성되면 그 객체를 사용
		- 객체 생성하는 코드는 없어야 함 
		
2. 설정파일
	- applicationContext.xml
		Person 객체 생성 관련된 spring 설정
		
3. 감 
	- servlet & jsp로 개발하는 servlet 객체들은 생성 자체를 container 내부에서 구현된 로직이 직접 생성 및 관리
	- container들을 servlet engine
		- 객체 생성 -> 관리 -> 소멸까지 모든 lifecycle 관리
	- tomcat은 제공받음(개발자는 사용)	
	- Spring framework은 일반 구조의 클래스들도 객체 생성, 관리 다 가능한 framework
		- container 라고 칭함
		- container 기능의 API는 framework 통해서 제공 단 생성은 직접 코드로 해야함
